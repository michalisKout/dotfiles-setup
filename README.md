# dotfiles-setup
A repository for setting up my dotfiles on a new equipment


## Instructions

1. Make script executable

_Mac Os_
```bash
chmod +x setup-macos.sh
```

_Windows (Linux subsystem)_
```bash
chmod +x setup-wsl.sh
```

2. Run script

_Mac Os_
```bash
./setup-macos.sh
```

_Windows (Linux subsystem)_
```bash
./setup-wsl.sh
```
